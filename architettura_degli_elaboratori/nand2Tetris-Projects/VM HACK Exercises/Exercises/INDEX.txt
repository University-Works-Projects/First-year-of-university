EXERCISE #1 V - Transale C code

    Write the relative VM HACK code of this C program:

    function mult (x,y){
        int sum, j;
        sum = 0;
        j = y;
        while (j != 0){
            sum = sum + x;
            j = j - 1;
        }
        return sum;
    }
